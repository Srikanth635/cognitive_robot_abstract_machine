# CRAM Documentation

Welcome to the Cognitive Robot Abstract Machine (CRAM) documentation hub.

### [pycram](./pycram/index.html)
**Python Cognitive Robot Abstract Machine**

The Python implementation of the Cognitive Robot Abstract Machine, providing tools for robot task planning and execution.

[Browse pycram Documentation →](./pycram/index.html)

### [semantic_digital_twin](./semantic_digital_twin/index.html)

CRAM is a cognitive architecture for robot control. For more information about the overall project, installation instructions, and contribution guidelines, please visit the [main repository](https://github.com/cram2/cognitive_robot_abstract_machine).
